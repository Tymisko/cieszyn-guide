# Cieszyn Guide

Mobile app developed in Flutter that is a guide to Cieszyn for tourists.

## Features
- Geolocation 
- Intuitive, responsive, user-friendly and visually appealing interface
- Saving favorite places
- Displaying statistics of visited places
